Prepare your environment#

  1. install the plugin prerequisites (from intellij documentation)
    1. git clone https://github.com/JetBrains/intellij-community.git
    2. install psiviewer plugin
    3. configure an IntelliJ Platform SDK
    4. attach the Community Edition source files to the SDK

Import#

  1. clone the plugin project
  2. import project from external model, choose gradle
  3. select use gradle 'wrapper' task configuration for easier setup
  4. Set project sdk to intellij platform sdk

Run#

You can launch a new idea instance with the run ide gradle task
